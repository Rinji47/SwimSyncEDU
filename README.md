# SwimSyncEDU

SwimSyncEDU is a Django-based web application for managing swimming classes, pools, trainers, and students.

## Features
- User authentication and dashboards for admin, trainers, and students
- Class and pool management
- Dynamic static assets (CSS & JS)
- MySQL database backend

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/Rinji47/SwimSyncEDU.git
   ```
2. Create a `.env` file in the project root for local secrets:
   ```env
   GOOGLE_MAPS_API_KEY=your_new_google_maps_browser_key
   PAYMENT_CALLBACK_BASE_URL=
   DJANGO_ALLOWED_HOSTS=
   ```
